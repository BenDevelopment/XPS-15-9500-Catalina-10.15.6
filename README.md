# XPS-15-9500-Catalina-10.15.6
A work in progress of getting OS X Catalina (10.15.6) to run on a Dell XPS 15 9500 using Opencore.

## Introduction
The EFI included in this repository was able to get Mac OS X Catalina 10.15.6 running on a Dell XPS 15 9500, albeit with many issues that will be reviewed in this guide. The specific laptop that this was tested on was the 3840x2400 touchscreen variant with a 1TB PC611 SK Hyinx NVME SSD.
